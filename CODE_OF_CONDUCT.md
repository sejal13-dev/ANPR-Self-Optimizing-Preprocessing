# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and inclusive environment for everyone.

## Expected Behavior

- Be respectful and professional.
- Welcome constructive feedback.
- Help others learn and grow.
- Respect different viewpoints and experiences.
- Use appropriate language in all communications.

## Unacceptable Behavior

- Harassment or discrimination.
- Offensive comments or personal attacks.
- Publishing private information without permission.
- Any behavior that creates an unwelcoming environment.

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards and may take appropriate action in response to unacceptable behavior.

## Contact

If you experience or witness unacceptable behavior, please open an issue or contact the repository maintainer.

Thank you for helping make this project welcoming for everyone.
