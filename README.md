# ANPR Self-Optimizing Preprocessing

This project is a preprocessing module for an Automatic Number Plate Recognition (ANPR) system.

## Features

- Vehicle Detection using YOLOv8
- License Plate Detection
- Four Image Preprocessing Strategies
- Self-Optimizing Preprocessing
- OCR using EasyOCR
- Character Correction
- Indian License Plate Validation

## Technologies Used

- Python
- OpenCV
- YOLOv8
- EasyOCR
- NumPy
- PyTorch
