# My Contribution

This repository contains **my individual contribution** to a team-based Automatic Number Plate Recognition (ANPR) project.

## My Responsibilities

- Designed and implemented the self-optimizing preprocessing pipeline.
- Developed multiple preprocessing strategies using OpenCV.
- Integrated EasyOCR for character recognition.
- Implemented OCR confidence comparison.
- Selected the best preprocessing strategy automatically.
- Applied character correction for common OCR errors.
- Improved recognition accuracy through image enhancement.

## Technologies Used

- Python
- OpenCV
- EasyOCR
- NumPy
- YOLOv8 (for integration with the detection pipeline)

## Note

Vehicle detection, API integration, and other modules were developed by different team members. This repository focuses only on my contribution.
