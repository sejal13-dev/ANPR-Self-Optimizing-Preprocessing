# Folder Description

## docs/

Contains all project documentation.

Includes:

- Architecture
- Installation Guide
- Usage Guide
- Project Structure
- Images used in documentation

---

## docs/images/

Contains diagrams used throughout the documentation.

Examples:

- Architecture Diagram
- Workflow Diagram
- OCR Pipeline
- Queue System
- Performance Evaluation

---

## models/

Stores trained YOLO models.

Example:

- yolov8n.pt
- plates_model_v1.pt

---

## sample_images/

Contains sample vehicle images used for testing.

---

## output/

Stores processed results generated by the application.

---

## src/

Contains the complete source code of the project.

Current file:

- main.py

---

## requirements.txt

Lists all Python dependencies required to run the project.

---

## README.md

Provides an overview of the project and setup instructions.

---

## LICENSE

Specifies the open-source license for this project.

---

## .gitignore

Defines files and folders that Git should ignore.
